# Laravel 9 Headless
## _React 18, modules SCSS, cURL WORDPRESS API v2_


#### Tech
- Vite
- Laravel 9
- React 18
- React Router 6
- Sass

#### Installation

Install the dependencies and devDependencies.

```sh
composer install
npm i
```

## Development

Up backend
```sh
php artisan serve
```
Up frontend
```sh
npm run dev
```
